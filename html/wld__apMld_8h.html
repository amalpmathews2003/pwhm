bool wld_apMld_fetchAffiliatedStaInfo ( wld_apMld_afStaInfo_t *  info,
int32_t  mldUnit,

)

Retrieve an affiliatedStaInfo for a given MAC Address associated to a given MLD unit.

Because inactive affiliated sta is remembered, and affiliatedSta mac addresses may be reused on other band it is possible there are multiple afSta with the same mac address, although there should never be more than one of them active at the same time.

Parameters
infothe struct in which the affiliated sta info will be written
mldUnitthe mld unit ID of the ap MLD.
macthe mac address for which to do a lookup for a matching affiliated sta.
Returns
  • true if an affiliated sta with the given mac address has been found. The info struct shall be properly filled in with the affiliated sta, the associated device to which the affiliated sta belongs, and the access point to which the associated device is associated.
  • false if no matching affiliated sta can be found, or some error took place. In this case, the info parameter will not be touched.
