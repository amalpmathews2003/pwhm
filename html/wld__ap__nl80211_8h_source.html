74/*
75 * @brief set a listener for all received nl80211 events from an AP interface.
76 *
77 * @param state pointer to nl80211 socket manager, on which socket the event would be received
78 * @param pAP pointer to accesspoint context
79 * @param pData user private data to provide when invoking handlers
80 * @param handlers Pointer to structure of callback functions, handling the received events
81 * Callbacks of unmanaged events shall remain null.
82 * Handlers will be called with pRef equal to pAP.
83 *
84 * @return SWL_RC_OK in case of success
85 * <= SWL_RC_ERROR otherwise
