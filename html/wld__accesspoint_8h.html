)

return the last assoc/reassoc frame sent by a station to a vap

Parameters
pAPthe accesspoint receiving the assoc/reassoc frame
macStationthe station mac address
datacontains the last assoc/reassoc tuple associated to macStation if found
Returns
- SWL_RC_OK: if the last assoc/reassoc frame is found for the given station
  • SWL_RC_INVALID_PARAM if pAP = NULL or macStation = NULL or data = NULL
  • SWL_RC_ERROR if no assoc/reassoc tuple associated to macStation is found
